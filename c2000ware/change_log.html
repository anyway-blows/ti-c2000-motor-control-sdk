What's New

3.04.00.00

  • CLB Type 3 Support - F2838x/F28002x
    • CLB driver updates for Type 3 features
    • CLB Tool updates for for Type 3 features
  • Dual Code Security Module (DCSM) Tool updates
  • F28002x Launchpad support
  • MCAN (CAN-FD) driver and examples on C28x - F2838x
  • Compiler migration for legacy devices (F2802x/3x/6x, F2823x, F2833x) to v20.2.1.LTS
  • Software Diagnostic Library (SDL) v2.01.00
  • Flash API libraries in EABI format are added for F2837xD, F2837xS and F28004x
  • New Examples
    • ERAD new examples - non-intrusive diagnostics on F28004x,F2838x and F28002x
    • FSI - PWM sync example on F28002x and F28004x
    • ACI motor control signal chain benchmark application on F2837xD and F28004x
