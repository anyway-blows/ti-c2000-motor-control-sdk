Introduction¶

C2000Ware for C2000™ real time control MCUs is a cohesive set of development software and documentation designed to minimize software development time. From device-specific drivers and libraries to device peripheral examples, C2000Ware provides a solid foundation to begin development and evaluation of your product. C2000Ware requires:

  • Code Composer Studio™ IDE (CCS) v9.3.0 or newer

  • C2000 Compiler v18.12.1 LTS or newer

  • ARM® Compiler v18.12.1 LTS or newer
